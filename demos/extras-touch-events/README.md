# extras-touch-events Demo

Demonstriert den `rv-touch-events`-Binder aus `@ribajs/extras`, der eine Berührungszone um Touch- und Gesten-Events wie `tap`, `doubletap`, `taphold`, `swipe` (in alle vier Richtungen) sowie `tapstart`/`tapend` anreichert. Alle ausgelösten Events erscheinen mit Koordinaten und Zeitstempel in einer Live-Konsole.

## Starten

```bash
git clone --recurse-submodules https://github.com/ribajs/riba.git
cd riba/demos/extras-touch-events
yarn install
yarn start
```
