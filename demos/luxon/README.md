# luxon Demo

Dokumentiert alle Riba.js-Formatter des `@ribajs/luxon`-Moduls für die Luxon-Datumsbibliothek. Es werden die Formatter-Präfixe `lx-` (DateTime), `lx-duration-` (Duration) und `lx-interval-` (Interval) mit Live-Ausgaben der jeweiligen Methoden wie `lx-now`, `lx-local`, `lx-utc` und `lx-from-seconds` veranschaulicht.

## Starten

```bash
git clone --recurse-submodules https://github.com/ribajs/riba.git
cd riba/demos/luxon
yarn install
yarn start
```
