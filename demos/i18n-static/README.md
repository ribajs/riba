# i18n-static Demo

Demonstriert das Internationalisierungssystem (`@ribajs/i18n`) mit statisch eingebetteten Übersetzungen für Deutsch und Englisch. Über einen Sprachumschalter lassen sich Texte, HTML-Inhalte, Eingabefeldwerte und Platzhalter eines Newsletter-Formulars live per Binding zwischen den Sprachen wechseln.

## Starten

```bash
git clone --recurse-submodules https://github.com/ribajs/riba.git
cd riba/demos/i18n-static
yarn install
yarn start
```
