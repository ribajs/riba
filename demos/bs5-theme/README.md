# bs5-theme Demo

Demonstriert das Light-/Dark-Theme-System von Riba.js mit Bootstrap 5, bestehend aus einer fixierten Navbar mit Hamburger-Menü, einer ausklappbaren Sidebar sowie Theme-Umschalter-Buttons. Die Seite reagiert automatisch auf das Betriebssystem-Theme und erlaubt dem Nutzer, zwischen Hell- und Dunkel-Modus zu wechseln.

## Starten

```bash
git clone --recurse-submodules https://github.com/ribajs/riba.git
cd riba/demos/bs5-theme
yarn install
yarn start
```
