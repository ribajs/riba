# rv-photoswipe Demo

Demonstriert die `rv-photoswipe`-Komponente als Riba.js-Integration der PhotoSwipe-Lightbox-Bibliothek. Es wird eine Bildergalerie mit Bildern von picsum.photos angezeigt, bei der ein Klick auf ein Vorschaubild die Vollansicht in einer Lightbox mit Navigations-, Zoom- und Vollbild-Steuerelementen öffnet.

## Starten

```bash
git clone --recurse-submodules https://github.com/ribajs/riba.git
cd riba/demos/rv-photoswipe
yarn install
yarn start
```
