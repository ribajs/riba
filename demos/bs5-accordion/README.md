# bs5-accordion Demo

Demonstriert die Bootstrap-5-Akkordeon-Komponente in zwei Varianten: einmal mit statischen `<template>`-Kindknoten und einmal durch dynamische Datenbindung über das `items`-Attribut sowie einen separaten `bs5-collapse`-Block.

## Starten

```bash
git clone --recurse-submodules https://github.com/ribajs/riba.git
cd riba/demos/bs5-accordion
yarn install
yarn start
```
