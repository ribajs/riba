Use of rv-bs5-attr-[breakpoint]-[attribute]

Set the attribute at the appropriate breakpoint

This also works for normal HTML elements

Use of rv-bs5-co-[breakpoint]-[attribute]

Pass the value to the component at the appropriate breakpoint

rv-bs5-attr vs rv-bs5-co

The difference between the normal attribute binder and the component attribute binder is that the attribute binder sets normal attributes and the component attribute binder passes the value directly to the component (without the detour via the attribute).

So, if larger data like an object should be passed to a component, then the ComponentAttributeBinder should be used, otherwise a very long JSON string would be passed as attribute.
