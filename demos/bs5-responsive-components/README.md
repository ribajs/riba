# bs5-responsive-components Demo

Zeigt die responsiven Bootstrap-5-Binder `rv-bs5-attr-[breakpoint]-[attribute]` und `rv-bs5-co-[breakpoint]-[attribute]`, mit denen Attributwerte (z. B. Icon-Quelle oder -Größe) abhängig vom aktiven Viewport-Breakpoint gesetzt werden. Zusätzlich wird eine Sidebar demonstriert, die auf kleinen Bildschirmen automatisch ausblendet und auf großen eingeblendet wird.

## Starten

```bash
git clone --recurse-submodules https://github.com/ribajs/riba.git
cd riba/demos/bs5-responsive-components
yarn install
yarn start
```
