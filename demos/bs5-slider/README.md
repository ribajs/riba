# bs5-slider Demo

Demonstriert den `bs5-slider` in drei Konfigurationen: mit Standardeinstellungen, mit benutzerdefinierten Einstellungen (2 Spalten, Drag, Infinite Loop) sowie mit Video-Slides und Autoplay – alle mit Vor-/Zurück-Steuerung und Indikatoren.

## Starten

```bash
git clone --recurse-submodules https://github.com/ribajs/riba.git
cd riba/demos/bs5-slider
yarn install
yarn start
```
